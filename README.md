# OIBSIP_python_task1
BMI CALCULATOR using python
Objective:
To develop a simple BMI Calculator web application that allows users to enter their weight and height, calculate Body Mass Index, and determine their health category accurately.

Tools Used:
HTML, CSS, and JavaScript for the front-end interface, and Python Flask framework for backend processing and handling user input.

Outcome:
The application successfully calculates BMI and displays the corresponding health category (Underweight, Normal, Overweight, or Obese) in a user-friendly web interface.
